# Theralevia — Your Daily Companion

A PWA for living well with chronic conditions. Built with React + Vite.

## Quick Start

```bash
# Install dependencies
npm install

# Run locally
npm run dev
```

Open `http://localhost:5173` in your browser.

## Deploy to Vercel (Fastest — Free)

1. Push this folder to a GitHub repo
2. Go to [vercel.com](https://vercel.com), sign in with GitHub
3. Click **"Add New Project"** → import your repo
4. Framework preset: **Vite** (auto-detected)
5. Click **Deploy**
6. Done — you'll get a URL like `theralevia.vercel.app`

## Deploy to Netlify (Also Free)

1. Push to GitHub
2. Go to [netlify.com](https://netlify.com), sign in
3. **"Add new site"** → **"Import an existing project"**
4. Build command: `npm run build`
5. Publish directory: `dist`
6. Deploy

## Deploy to GitHub Pages (Free, No Account Needed Beyond GitHub)

1. In `vite.config.js`, add `base: '/<repo-name>/'` to the config
2. Run `npm run build`
3. Push the `dist` folder to a `gh-pages` branch
4. Enable Pages in repo Settings → Pages → Source: `gh-pages`

## Install on Android

Once deployed to any of the above:

1. Open the URL in **Chrome** on the Android device
2. Tap the **three-dot menu** (⋮) in the top right
3. Tap **"Add to Home screen"** or **"Install app"**
4. The app icon appears on the home screen and launches full-screen

> Chrome may also show an automatic "Add to Home Screen" banner
> after the second visit within a few minutes.

## Install on iPhone

1. Open the URL in **Safari**
2. Tap the **Share** button (square with arrow)
3. Tap **"Add to Home Screen"**

## Project Structure

```
theralevia/
├── index.html          # Entry HTML with PWA meta tags
├── package.json        # Dependencies (React, Vite, PWA plugin)
├── vite.config.js      # Vite + PWA manifest + service worker config
├── public/
│   ├── favicon.svg     # Browser tab icon
│   ├── icon-192.png    # PWA icon (homescreen)
│   └── icon-512.png    # PWA icon (splash screen)
└── src/
    ├── main.jsx        # React entry — initializes storage shim
    ├── storage.js      # localStorage adapter (replaces window.storage)
    └── App.jsx         # Full application component
```

## Replacing the Placeholder Icons

The included icons are basic text-based placeholders. To create polished ones:

- Use [favicon.io](https://favicon.io) or [realfavicongenerator.net](https://realfavicongenerator.net)
- Replace `icon-192.png` and `icon-512.png` in `public/`
- The 512px icon should have `purpose: "any maskable"` padding (safe zone)

## Notes

- All data is stored in `localStorage` on the device — nothing leaves the phone
- The service worker (auto-generated by `vite-plugin-pwa`) caches the app
  for offline use after the first visit
- Google Fonts (Lora + DM Sans) are cached for 1 year after first load
- The community feature uses local storage only — for real multi-user
  support you'd need a backend (Firebase Realtime DB is the easiest path)
